# Getting Started

Use Database Support AI to investigate production database issues with evidence.

Navigation path: Signup or Login -> Workspaces -> Database Connections -> Documents -> AI Chat.

Steps:
1. Create or select a workspace.
2. Add a database connection.
3. Test the connection.
4. Upload runbooks, known issues, and technical notes.
5. Ask an investigation question in AI Chat.
6. Review the answer, evidence, report, and verification checks.
7. Run only safe suggested verification checks.
8. Download the generated report.

Warnings:
- The investigation AI is evidence-grounded but still needs human review.
- Verification SQL is read-only.
- Do not apply fixes without DBA or lead approval.
