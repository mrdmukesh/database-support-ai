# Workspace Setup

Workspaces isolate databases, documents, investigations, reports, and learning records.

Navigation path: Workspaces -> Create workspace.

Steps:
1. Open Workspaces.
2. Enter a workspace name.
3. Enter a workspace slug.
4. Click Create workspace.
5. Use this workspace when adding connections, documents, and investigations.

Warnings:
- Use separate workspaces for separate customers, environments, or systems.
- Do not upload one customer's documents into another customer's workspace.
