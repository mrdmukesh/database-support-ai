# Audit Logs

Audit logs record important security and investigation actions.

Navigation path: Database administrator query against the app database audit_logs table.

Tracked examples:
- USER_LOGIN
- INVESTIGATION_STARTED
- VERIFICATION_SQL_EXECUTED
- REPORT_DOWNLOADED
- database connection changes
- document uploads
- knowledge approvals

Warnings:
- Audit logs are for governance and troubleshooting.
- Access to audit logs should be limited to admins and auditors.
