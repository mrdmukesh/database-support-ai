# Asking Investigation Questions

Good questions include the business key, symptom, expected behavior, and what you want investigated.

Navigation path: AI Chat -> Question -> Ask AI.

Steps:
1. Select the workspace.
2. Describe the issue.
3. Include identifiers such as order number, appointment number, ticket id, or container id.
4. Ask for affected object, parent object, evidence, write path, root cause, fix, tests, proof of fix, and rollback.
5. Click Ask AI.

Example:
Appointment APT-2005 created two active lab orders. Investigate like a Senior Production Support Engineer.

Warnings:
- AI Chat investigates database issues; Help Assistant explains how to use the app.
- Unsafe or destructive SQL requests are rejected.
