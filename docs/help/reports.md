# Reports

Reports summarize investigation evidence, reasoning, verification, recommended fix, tests, proof of fix, and rollback.

Navigation path: AI Chat -> Investigation Complete -> Download PDF, Word, Excel, or View HTML Report.

Steps:
1. Run an investigation.
2. Wait for Investigation Complete.
3. Use the report download buttons.
4. After running verification checks, use the verified report buttons in Suggested Verification Checks.

Warnings:
- Reports are based on collected evidence and may still need DBA review.
- Download access is protected by workspace permissions.
